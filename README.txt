To run the following please follow these steps
1) Download XAMPP
2) type localhost/phpmyadmin in your preffered browser
3) click on Server: localhost on the top left side of the screen
4) go to import-->choose file-->select 'localhost.sql'
5) exit the browser and locate the folder containing htdocs
    MAC
    Applications --> XAMPP -->htdocs 
    WINDOWS
    C:-->Xampp-->htdocs
6) Delete all files in the htdocs folder and add 'Enactus'
7) Open your web browser and type localhost
8) go to Enactus-->main.php

NOTE: Following can be hosted on a local device using MAMP, but I dont like to leak my IP ;)
